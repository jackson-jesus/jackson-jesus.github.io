Início

Oi, escrevo este site, apelidado de sitezinho, para tentar organizar conteúdos relacionados com estudos pessoais sob a licença Creative Commons (link externo). Sua versão anterior continua disponível neste link (externo), assim como a página de referências (link externo).

Esta nova versão é gerada com o plugin Vimwiki (link externo) do programa Vim (link externo), meu editor de texto favorito. O motivo é simples: praticidade. Depois de alguns ajustes, é só digitar, exportar uma única vez e apenas ir salvando as edições.

O sitezinho mantém uma cópia no Vaporhole (link externo), uma rede social lenta e também está listado no Lerama (link externo), um diretório e buscador de blogs pessoais atualizados em tempo real; dois excelentes lugares para se ler textos feitos por humanos.

Relembrando, aqui não coleto dados pessoais, não forço anúncios, manipulação algorítmica, enfim, nada de ruim que se popularizou pela Internet nos últimos tempos. O único código JavaScript usado coleta estritamente métricas de acesso e você pode saber mais nesta página do Simple Analytics (link externo).
